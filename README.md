# Stephen Heibel - Personal Portfolio

This is a single-page static portfolio website built using HTML, CSS, and JavaScript. It was designed to showcase my skills, projects, and professional recommendations in a clean, responsive format.

## Features
* **Navigation Bar**: A sticky navigation bar with custom hover effects for easy access to different sections.
* **About Me**: A brief introduction and profile picture.
* **Skills**: A grid layout displaying core technical skills alongside FontAwesome icons.
* **Projects**: A section detailing recent development projects.
* **Recommendations**: A dynamic section featuring existing recommendations, plus a form that allows visitors to submit new recommendations. Submissions use a JavaScript confirmation dialogue and update the list in real-time.

## How to View
To view this portfolio, you do not need to install any server software. It runs entirely in the browser.

1. Navigate to the project folder on your local machine.
2. Double-click the `index.html` file.
3. The website will open and function in your default web browser.

## Project Structure
* `index.html`: The main structural layout and content of the webpage.
* `style.css`: The stylesheet governing the visual presentation, layout, and animations.
* `script.js`: The JavaScript file handling interactivity, specifically the recommendation submission form.
* `OptimusGarth.jpg`: The profile image used in the "About Me" section.
* `README.md`: This documentation file.
